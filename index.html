Precision for Web Developers

Because sometimes developers need that extra level of precision for measurements.

Get Started

Just drag this link to your bookmarks: Guides.js. Click it once to enable. Click it again to toggle guides visibility.

Usage

H - Add a horizontal guide
V - Add a vertical guide
ESC - Cancel adding a guide
C - Clear all guides
L## - Link two guides (e.g. add two vertical guides, then type 'L12')
